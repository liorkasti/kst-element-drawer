# KST Elements Drawer app

### Installation:

1.  Clone the project to your local machine:
    ```sh
    git clone https://github.com/liorkasti/kst-element-drawer
    ```
1.  Install the application dependencies:
    ```sh
    npm install `or` yarn
    ```
1.  Run the app
    ```sh
    npm start `or` yarn start
    ```

Enjoy! thank you.
